Personalised Dustbin
Arduino Based Electronics Project

To develope a simple but useful project of personalized dustbin using Arduino and ultrasonic sensor along with humidity & temperature sensor. Using this project, the lid of the dustbin stays closed, so that waste is not exposed (to avoid flies and mosquitos) and when you want dispose any waste, it will automatically open the lid. It will also automatically segregate the waste in organic and non-organic waste. When an obstacle is stuck on the sensor, it may give a reading which may vary from the actual reading leading to abrasive action from microcontroller which is not necessary.
